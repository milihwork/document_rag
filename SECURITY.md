# Security policy

## Reporting a vulnerability

If you believe you have found a security vulnerability in this project, please report it responsibly:

- Use GitHub private reporting: open a draft security advisory from this repository’s **Security** tab → **Advisories** → **New draft**.
- Do not open a public issue, discussion, or pull request for security reports.

We will acknowledge your report and respond within a reasonable time. We ask that you allow time for a fix before any public disclosure.

Thank you for helping keep this project safe.
